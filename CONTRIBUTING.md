## Adding a New Project

If you want to add a new project under Labs:

1. Create a new directory under `projects/`
2. Include a `README.md` with:
   - Project description
   - Goal / problem it solves
   - Maintainer(s)
   - Current status (idea / prototype / active / archived)
3. Keep projects lightweight and focused
4. Hackathon and experimental projects are welcome

Not all projects are expected to be long-lived â€” learning is a valid outcome.

## Contribution Workflow

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a Pull Request (PR)
5. Engage in discussion and iterate if needed

Maintainers will review PRs with a focus on collaboration and learning, not perfection.
